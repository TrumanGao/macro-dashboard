# macro-dashboard

1. find a good-looking macroeconomic dashboard website as a design template to imitate and develop. For example, [Trading Economics](https://tradingeconomics.com/), [FRED](https://fred.stlouisfed.org/), [National Bureau of Statistics of China 国家统计局](http://www.stats.gov.cn/), [CEIC Data](https://www.ceicdata.com/zh-hans/), [Wind](https://www.wind.com.cn/), [East Money 东方财富](https://data.eastmoney.com/), [JoinQuant 聚宽](https://www.joinquant.com/data/dict/macroeconomics), [RiceQuant 米筐](https://www.ricequant.com/home/macro), [iResearch 艾瑞咨询](https://www.iresearch.com.cn/), etc.

2. 列举北美前端开发流行的技术栈和工具：
   - 语言：TypeScript
   - 前端框架：React
   - 状态管理：zustand
   - 路由管理：React Router
   - 请求库：Axios, React Query
   - 样式处理：Tailwind CSS
   - UI 组件库：Radix UI, Shadcn UI
   - 数据可视化：D3.js, Recharts
   - 构建工具：Vite
   - 代码质量：ESLint, Prettier
   - 版本控制：Git, GitHub
   - node 版本管理：Volta
   - 包管理器：pnpm
